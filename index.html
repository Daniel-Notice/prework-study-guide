HTML

  • The body represents the visible content shown to the user.
  • The base element specifies the base URL and/or target for all relative URLs in a docuemnt.

CSS

  • The display property can be assigned with the block value, which assigns each image to start on a new line.
  • The box shadow CSS property adds shadow effects around an elements frame.
  • The period before the class name in CSS is used as a class selextor.
