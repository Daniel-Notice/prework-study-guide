# Prework Study Guide Webpage

## Description


The purpose of this study guide is to aid bootcamp students through the prework stage. It covers topics such as HTML,CSS,Git and Javascript. While creating this webpage, I have learnt about all four topics and made notes for many to see.

## Usage

To use this study guide, pick a topic to review. To help you make a decision on what to have a look at first, use the dev tools. To access dev tools, use the f12 button and then navigate to the console tab. You will then see a list of topics to choose from.
